# Introducción

Markdown es un lenguaje de marcado ligero que permite formatear texto de forma **sencilla** y _elegante_.

## Listas

### Lista desordenada

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

### Lista ordenada

1. Primer paso
2. Segundo paso
3. Tercer paso
